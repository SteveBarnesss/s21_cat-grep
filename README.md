# s21_cat & s21_grep

Реализация утилит командной строки **`cat`** и **`grep`** на языке C.

Проект повторяет поведение стандартных Unix-утилит и предназначен для изучения работы с файлами, строками и аргументами командной строки.

## 📌 О проекте

В рамках проекта реализованы:

* `s21_cat` — аналог утилиты `cat`
* `s21_grep` — аналог утилиты `grep`

Поддерживаются основные флаги и их комбинации (частично соответствуют GNU-реализации).

📎 Поддерживаемые ОС:

* macOS
* Linux *(возможны отличия в поведении тестов)*

## 📖 Использование

### 🔹 s21_cat

Выводит содержимое файлов.

```bash
./s21_cat [options] [file...]
```

### Поддерживаемые опции:

| Флаг | Описание                                      |
| ---- | --------------------------------------------- |
| `-b` | нумерует только непустые строки               |
| `-e` | отображает `$` в конце строки (включает `-v`) |
| `-n` | нумерует все строки                           |
| `-s` | удаляет повторяющиеся пустые строки           |
| `-t` | отображает табы как `^I` (включает `-v`)      |

---

### 🔹 s21_grep

Ищет строки, соответствующие шаблону.

```bash
./s21_grep [options] pattern [file...]
```
| Флаг   | Описание                                         |
| ------ | ------------------------------------------------ |
| `-e`   | задать шаблон поиска                             |
| `-f`   | читать шаблоны из файла                          |
| `-i`   | игнорировать регистр                             |
| `-v`   | инвертировать поиск (вывод строк без совпадений) |
| `-c`   | вывести количество совпадений                    |
| `-l`   | вывести только имена файлов с совпадениями       |
| `-n`   | показать номер строки                            |
| `-h`   | скрыть имя файла в выводе                        |
| `-s`   | подавить сообщения об ошибках                    |
| `-o`   | вывести только совпавшую часть строки            |
| `-A N` | вывести N строк после совпадения                 |
| `-B N` | вывести N строк до совпадения                    |
| `-C N` | вывести N строк вокруг совпадения                |
| `-E`   | использовать расширенные регулярные выражения    |
| `-F`   | искать                                           |


| Комбинация | Что делает                               |
| ---------- | ---------------------------------------- |
| `-iv`      | игнорировать регистр + инверсия          |
| `-in`      | регистр игнорируется + номера строк      |
| `-rn`      | рекурсивно + номера строк                |
| `-rin`     | рекурсивно + без регистра + номера строк |
| `-nC 3`    | номер строки + 3 строки контекста        |
